## API auditoria

**Find by Campo:** *Post*

``http://localhost:8080/auditoria/find/campo``

>*Json for Insert*

```
{
    "tabela": "usuario",
    "campo": "nome",
    "dataInicio": "2024-01-01T00:00:00",
    "dataFim": "2024-12-31T23:59:59"
}
```
